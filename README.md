# Caviar Network

Run Web:

1. chnage directory to web : ``cd caviar/web``
2. install dependencies : ``composer install``
3. grant access to apache user : ``sudo chmod -R 777 storage/``
4. install docker & docker-compose 
5. run : ``sudo ./vendor/bin/sail up``
